## 2. Architecture Improvements

### 2.1 constants.py — Novel-Agnostic

**Before:** Hardcoded `global-descent` in every path.
**After:** `get_novel_root(slug)` with env var `NOVEL_SLUG` fallback.

```python
# Before
NOVEL_ROOT = TOOLS_DIR.parent / 'novels' / 'global-descent'

# After
NOVEL_ROOT = NOVELS_DIR / os.environ.get('NOVEL_SLUG', 'global-descent')
# + helper: get_novel_root(slug), get_glossary_dir(slug), etc.
```

### 2.2 save_chapter.py — Unified Tool

**Before:** `save_chapter.py` (MD-only) + `save_json.py` (JSON-only) — overlap.
**After:** Single `save_chapter.py` — reads .json (canonical), .md fallback, schema + doctor + FTS.

### 2.3 validate_chapter.py — JSON-Aware

**Before:** Read `.md` only → silently failing on all 52 chapters.
**After:** Reads `.json` blocks, extracts TH text, validates correctly.

### 2.4 tools/__init__.py — Package Marker

Added `__init__.py` so `tools/` is a proper Python package.
