# Glossary Index — 全球降臨：帶著嫂嫂末世種田

> Master view of the glossary. For translation work, **read `locked.md` first
> (mandatory), then `reference.md` (for context), and only `auto.md` if you
> encounter a term that's not in the first two.**

## How to use

- **Always read `locked.md`** — these terms are mandated by `style.md` and the main cast. NEVER use a different Thai version.
- **Read `reference.md` for context** — recurring NPCs, common items, common skills. The reader is more consistent if you use these exactly.
- **Only consult `auto.md` for specific terms** — one-off names, unique items. If a term is in source but not in `auto.md`, suggest a Thai version and append to `auto.md` + this index.
- **On disagreement** — `locked.md` wins over `reference.md` wins over `auto.md`.

## Adding new terms

When you encounter a new term in source that doesn't appear in any of the 3 files:

1. Decide the bucket:
   - **Locked** if it's a main cast, style.md-mandated term, or core game system
   - **Reference** if it appears 3+ times across chapters (or you strongly expect it will)
   - **Auto** otherwise
2. Add the term to the appropriate file in the table format
3. Update the count in this index

## Cross-bucket conflicts

If a term already has a different Thai version somewhere (e.g., old `ice-apocalypse` glossary had 极地人 = ชาวโพลาร์, but style.md says คนเมืองหนาว), the bucket with the higher priority wins. Style.md wins for in-world terms.
